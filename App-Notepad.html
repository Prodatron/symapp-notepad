1000-Introduction


INTRODUCTION TO NOTEPAD

Notepad is a text editor for SymbOS; it creates and edits plain text documents with sizes of up to 16KB. It is mainly based on the multiline text input control of the SymbOS GUI (Desktop Manager) to demonstrates its powerful features but extends it with additional functions like search and replace as well. First released in 2013 Notepad has been part of every version of SymbOS ever since.

1100-Feature overview


FEATURES

- cut/copy/paste/delete all mouse and keyboard based
- several short cuts for jumping between words/lines/text blocks etc.
- find/replace
- several proportional and non-proportional fonts are selectable
- optional auto word wrap
- lines can have lengthes of up to 16000 chars
- tab stops
- insert time/date
- free pen/paper colour definition

2000-Using Notepad


USING NOTEPAD

Notepad is a complete implementation of a usual text editor known from all modern graphical user interfaces. You can use the mouse and the keyboard in the same way like you do it on your PC, like for text marking and text editing.

2100-Screen Layout


SCREEN LAYOUT

Notepads screen layout is as simple as possible. At the top of the window you have the pull down menus providing most of the features sorted by categories.

The main window is just showing the edited text.

At the bottom of the window you have an optional status bar which is showing the current cursor position inside the document, the length of the whole document and the length of the marked area, if anything is marked.
